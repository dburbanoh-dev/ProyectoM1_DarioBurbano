# 🎨 Colorfly Studio — Generador de Paletas de Color

 **ProyectoM1_DarioBurbano**
MVP funcional de aplicación web estática e interactiva  
Módulo 1 · Desarrollador Frontend Junior

## 📋 Descripción

Herramienta web estática que permite a los equipos de diseño de
**Colorfly Studio** generar paletas de colores aleatorias de forma rápida e intuitiva. Desarrollada con HTML, CSS y JavaScript — sin dependencias externas.

## Funcionalidades implementadas

| Requisito | Estado |
| Botón "Generar paleta" operativo | ✅ |
| Generación correcta de colores aleatorios | ✅ |
| Render dinámico según tamaño seleccionado (6 / 8 / 9) | ✅ |
| Formatos HEX y HSL | ✅ |
| Visualización de código de color junto a la muestra | ✅ |
| Microfeedback visible (toast + badge "Copiado") | ✅ |
| HTML semántico | ✅ |
| Accesibilidad: labels asociados, contraste, foco visible | ✅ |
| Funcionamiento en desktop | ✅ |
